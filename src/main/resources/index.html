Kotlin Spring 2.0 Boilerplate Example Microservice

How to access the documentation?

This application provides an interactive Swagger documentation that you can access here.

How to access the source code?

The source code is accessible on GitHub at the following address: https://github.com/erik1066/ex-ms-kotlin-boilerplate.

Quick Start

This microservice app is primarily designed as a demo of Kotlin, Swagger and Spring Boot

To test use the following URL (using Swagger or curl):

curl -X GET \
  --header 'Content-Type: application/json' \
  --header 'Accept: application/json' \
  'http://127.0.0.1:8080/greeting?name=Kotlin'
